# 术语详解表

### 1. AI与大模型技术栈 (AI & LLM Tech Stack)

| 英文缩写/单词                | 全称 (Full Name)               | 中文释义与简历中语境解释                                                                                 |
| :--------------------------- | :----------------------------- | :------------------------------------------------------------------------------------------------------- |
| **LLM**                | Large Language Model           | **大型语言模型**。指GPT-4、Claude等具有强大理解和生成能力的AI模型。                                |
| **Agent**              | -                              | **智能体**。具备感知、决策、执行能力的AI系统（如您的“数据分析助手”）。                           |
| **MCP**                | Model Context Protocol         | **模型上下文协议**。一种标准化的AI连接协议，让AI能安全地连接本地数据和工具（您项目的核心技术点）。 |
| **RAG**                | Retrieval-Augmented Generation | **检索增强生成**。让AI先查知识库再回答问题，解决AI胡说八道的问题（用于您的供应链知识库项目）。     |
| **CoT**                | Chain of Thought               | **思维链**。引导AI一步步推理的提示词技术（如“请先分析原因，再给出方案”）。                       |
| **LangChain**          | -                              | **AI开发框架**。用于构建大模型应用的开源框架，常用于把LLM和外部工具串联起来。                      |
| **Dify**               | -                              | **开源LLM应用开发平台**。您用于快速搭建AI知识库和工作流的平台。                                    |
| **Copilot**            | -                              | **副驾驶/助手**。指辅助人类工作的AI工具（如“研发效能Copilot”）。                                 |
| **Prompt Engineering** | -                              | **提示词工程**。设计高效指令让AI输出高质量结果的技巧。                                             |

### 3. 业务与财务指标 (Business & Finance Metrics)

| 英文缩写/单词   | 全称 (Full Name)                                      | 中文释义与简历中语境解释                                                           |
| :-------------- | :---------------------------------------------------- | :--------------------------------------------------------------------------------- |
| **PAC**   | Product Acceptance Certificate                        | **产品验收证书**。项目交付或产品上市的重要里程碑。                           |
| **TTM**   | Time To Market                                        | **上市时间**。产品从构思到推向市场所需的时间（研发效能的关键指标）。         |
| **SPU**   | Standard Product Unit                                 | **标准化产品单元**。属性值、特性相同的商品就可以称为一个SPU（如iPhone 15）。 |
| **WIP**   | Work In Progress                                      | **在制品/在研项目**。正在进行中但尚未完成的工作或库存。                      |
| **N+3**   | Rolling Forecast (N+3 months)                         | **滚动预测**。预测当前月(N)及未来3个月的需求或计划。                         |
| **DOA**   | Dead On Arrival                                       | **到货即损**。产品刚送到客户手中就是坏的（质量核心指标）。                   |
| **NDF**   | No Defect Found                                       | **未发现故障**。客户退回产品但检测没问题（反映售后成本浪费）。               |
| **ROI**   | Return On Investment                                  | **投资回报率**。衡量项目投入产出比的核心财务指标。                           |
| **GMV**   | Gross Merchandise Volume                              | **商品交易总额**。电商或平台类业务的成交总金额。                             |
| **MAU**   | Monthly Active Users                                  | **月活跃用户数**。衡量产品用户粘性的关键指标。                               |
| **AARRR** | Acquisition, Activation, Retention, Revenue, Referral | **海盗模型**。用户生命周期模型（获取、激活、留存、变现、推荐）。             |
| **RFM**   | Recency, Frequency, Monetary                          | **用户价值模型**。通过最近一次消费、消费频率、消费金额来划分用户等级。       |
| **T+1**   | Transaction + 1 day                                   | **T+1结算**。交易发生后的第二个工作日完成数据处理或资金结算。                |
